# 🛒 Origin E-commerce

Sistema de e-commerce desenvolvido utilizando Java e Spring Boot para estudos de desenvolvimento Back-End.

---

## 🚀 Tecnologias Utilizadas

- Java
- Spring Boot
- Jpa Data
- JWT Auth0
- SpringDoc
- MySQL
- Maven

---

## 📌 Funcionalidades

✔ Cadastro de usuários  
✔ Login e autenticação  
✔ Cadastro de produtos  
✔ Listagem de produtos  
✔ Carrinho de compras  
✔ Gerenciamento de pedidos  

---

## 🎯 Objetivo do Projeto

Projeto criado com o objetivo de aprimorar conhecimentos em desenvolvimento back-end, banco de dados, APIs REST e estruturação de sistemas web.
